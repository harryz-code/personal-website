# Personal Portfolio

A simple, single-page portfolio with a dark theme and warm accent colors.

## Quick start

Open `index.html` in a browser, or run a local server:

```bash
# Python 3
python3 -m http.server 8000

# Node (if you have npx)
npx serve .
```

Then visit `http://localhost:8000`.

## Customize

1. **Content** — Edit `index.html`: replace "Your Name", tagline, about text, project titles/descriptions, and contact links.
2. **Colors** — In `styles.css`, change the `:root` variables (e.g. `--accent`, `--bg`, `--text`).
3. **Projects** — Add or remove `.project-card` items and point the links to your live sites or repos.

No build step required; pure HTML, CSS, and vanilla JavaScript.
